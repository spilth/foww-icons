FOWW ICN-SVG

Wasteland Warfare Icons

This is a collection of icons for the game Fallout: Wasteland Warfare created by Modiphius Entertainment. They are provided in the Scalable Vector Format (SVG) so that they can be used at any size without losing quality.

About

This project is created and maintained by Brian Kelly. You can view the project source code on GitHub.

If you notice a problem or would like to contribute to the project, please open an issue.
